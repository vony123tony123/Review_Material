# 保險買賣網站
本專案為中央大學資訊管理學系（IM3069-*）系統分析與設計課程之期末專案，本專案使用Eclipse EE作為主要IDE，採用Apache Tomcat作為網站伺服器，Oracle MySQL作為資料庫。
